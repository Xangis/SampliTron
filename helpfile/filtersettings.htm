Filter Settings Dialog

The filter settings dialog is shown by clicking on the cone filter icon in the main window.

The filter type can be low-pass, high-pass, band-pass, or notch.

The Filter Frequency setting controls the center frequency for the filter.

The Filter Q setting controls the bandwidth of the filter.

The Filter Enabled button checkbox enables or disables the filter.

When you are done changing filter settings, click "OK" to exit this window.
