SampliTron
MIDI Settings Dialog

The MIDI settings dialog is shown by clicking on the MIDI connector icon in the main program window.

The MIDI out device is the device that will be used for sending any and all MIDI messages if MIDI output is enabled. Select an item from the drop-down list to change the active MIDI output device.

The MIDI in device is the device that will be used to control SampliTron. Select an item from the drop-down list to change the active MIDI output device.

The MIDI input channel is the channel number to listen on for MIDI messages. Valid settings are 1 to 16.

The MIDI output channel is the channel that MIDI messages will be sent on. Valid settings are 1 to 16.

The MIDI output enabled indicator shows whether outgoing MIDI messages will be sent. Click this to change the setting. If this is gray (off), no outgoing MIDI messages will be sent and SampliTron will not be usable as a MIDI controller.

When you are done changing MIDI settings, click "close" to exit this window.
